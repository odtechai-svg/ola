# OLA Route Map

- `/` landing / prototype intro
- `/onboarding` source language + target language selection
- `/home` Today screen
- `/blocks` block map and progression
- `/session/[sessionId]` session player
- `/progress` learner metrics
- `/settings` preferences and permissions

## Critical route notes
- Session should always be directly accessible from Today.
- Onboarding must enforce source language first, target language second.
- The block map needs four visual curricular phases in production.
